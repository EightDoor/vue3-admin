# 技术栈
- typescript + vue3 + composition-api + ant-design-vue2.x + vue-router@4.x + vuex@4.x

# 完成功能

- 用户管理 <> 关联角色
- 角色管理 <> 关联菜单
- 菜单管理
- 部门管理
- 字典管理
- 登录、权限校验
- 动态路由生成 - 按钮权限
- 菜单栏切换面包屑
- tab展示历史导航路由
# 运行
- 安装依赖 `pnpm install`
- 运行 `npm run dev`

## 按钮权限使用
- ```vue
  // 使用的是按钮权限的名字
  <a-button type="danger" v-bt-auth:del></a-button>
  // 或者
  // 使用的是自定义的名称  '删除 - 1'
  <a-button type="danger" v-bt-auth:del="{title:true}">删除 - 1</a-button>
  ```
